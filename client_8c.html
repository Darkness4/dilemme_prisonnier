Detailed Description

Joueur pouvant communiquer avec d'autres clients/serveur.

Connection via IP, Port et Pseudo.

DNS supporté.

Fonctionnalités :

  • Lecture P2P Client-Serveur
  • Ecriture P2P Client-Serveur

Usage:

./client <ip> <port> <pseudo>
